二学期第一次周报

发表于 2019-03-23

1.本周学习方向：

CLion安装与配置

2.学习参考：

CLion官网

3.总结：

       CLion是由捷克著名软件开发商JetBrains公司开发的专业 C/C++ IDE，以IntelliJ为基础设计，包含了许多智能功能来提高开发人员的生产力，提高开发人员的工作效率。
       我也是才开始使用这个IDE，但我一用上便已喜欢上了它，其优秀的智能提示和代码补全功能比之 VS 可谓有过之而无不及，并且还免去了 VS 臃肿的缺陷，可谓是 C/C++ 高效开发的利器。但就是在Windows下安装MinGW编译环境比较麻烦，但有一个便利的方法就是通过 Dev C++ 来快捷获得可用编译环境，只需将库和编译器的目录设置成 Dev C++ 的，便可达到快速搭建环境的目的。还有CLion官方原生只有英文版，但通过社区提供的汉化包能达到80%以上的汉化率，不影响使用。另外，CLion其实是收费的，但学生可以通过校园邮箱来申请教育账户来免费使用该软件。CLion 还有很多高级功能我还没有发现，我也希望大家也能享受到这一神器带来的好处。

4.LeetCode：

4.1.验证回文字符串
之前做了很多次都失败了，具体的步骤我忘了，大致的是打算本地操作，把所有其他字符都给去掉，然后再判断回文，但因过于复杂，所以都失败了。不久前才想到可以建立一个新的字符数组，把原数组遍历后的合法字符提取过来再判断，终于成功了。

4.2.最长公共前缀
我写的第一个较为完整的Java函数，因为之前用C一直没做出来，并且在学Java的过程中发现Java对字符串的才做更为灵活，才打算用Java做的。通过对substring方法的正确使用，做出来了。

4.3.Fizz Buzz
之前因为对returnSize的真实意义理解有偏差，所以导致如此一个简单的题一直没通过。经过大佬指点明白了，但仍然认为returnSize有些多余，因为他完全可以在主函数中被赋值，完全不需要传到子函数中。

4.4.删除链表中的节点
简单，只需把next指向next的next就行了。

4.5.删除链表的倒数第N个节点
自己写的实现始终是失败的，所以看了网上大佬的写法。这才令我明白了什么是双指针法，不，准确来说是三指针法。第一个指针用来将倒数转化为正数，第二、三个指针才用来删除指针，再考虑到特殊情况就通过了。

4.6.合并两个有序链表
看的网上的实现，通过递归，逐步将两个链表中的所有元素从小到大重新组合成一个新链表。

研究数据结构。
